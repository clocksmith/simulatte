## Code Agent

**Prime Directive:** Write JavaScript/WebGL code for interactive simulations and visualizations.

### Before Starting
- Read `README.md` for project structure and acronym pairs
- Read `EMOJI.md` for approved Unicode symbols

### Guardrails
- Enforce `EMOJI.md`; use only approved Unicode symbols, no emojis
- Maintain acronym reciprocity (update both halves of pairs)
- Test numerical accuracy for physics simulations
- Optimize for browser performance (WebGL/WebGPU)
